# 🏥 MediScan AI
### AI Powered Medical Report Analyzer

MediScan AI helps patients understand 
their medical reports in plain English 
using NLP and BERT models.

---

## 🌟 Features

- 📄 Upload PDF medical reports
- 🔍 Auto detects report type
  (Blood Test / Prescription / Discharge)
- ✅ Normal / Abnormal value detection
- 💬 Plain English explanation of results
- ⚠️ Risk level indicator
  (Low / Medium / High)
- 🤖 Ask your report — chat with your report
- 📊 Visual dashboard with charts
- 🕓 Report history tracking

---

## 🧰 Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript, Chart.js |
| Backend | Python, Flask |
| NLP | spaCy, HuggingFace Transformers |
| BERT Models | DistilBERT, BART |
| Database | MySQL |
| PDF Reading | pdfplumber |
